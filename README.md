# High-voltage-differential-probe
Parameter requirements:
Input range: ± 1250V
Signal bandwidth: 100MHz
Input power supply 5V isolation, switchable battery power supply and charging/discharging using domestic components, integrated isolation transformer tap to achieve ± 5V, has passed high voltage and insulation withstand voltage tests, and has undergone 5M square wave and 20M sine wave signal tests. The overall signal testing accuracy of the probe is about ± 2%
